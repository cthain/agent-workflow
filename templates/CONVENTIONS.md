# CONVENTIONS.md

Read and follow `AGENTS.md`.

This file exists as a thin adapter for tools that prefer a generic conventions file.

Canonical project instructions live in:

```text
AGENTS.md
```

Active task context lives in:

```text
.planning/current/task-plan.md
.planning/current/notes.md
```

Use hyphens, not underscores, in file and directory names.
