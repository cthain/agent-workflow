# Prompt: Multi-agent handoff

Use this when handing work from one agent or mode to another.

```text
Prepare a handoff note in `.planning/current/notes.md`.

Include:

- current state
- completed work
- remaining work
- known risks
- commands already run
- commands still needed
- files likely needing attention
- suggested next step

Do not overwrite existing notes. Add a new handoff section.
```
