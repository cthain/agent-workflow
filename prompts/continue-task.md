# Prompt: Continue active task

Use this when returning to an active task.

```text
Read:

- `AGENTS.md`
- `.planning/current/task-plan.md`
- `.planning/current/notes.md`

Then inspect the repository before making code changes.

Update the plan if the current repository state contradicts or changes the plan.

Continue the task in small, understandable steps.

Before finishing, update the planning files and summarize:

- what changed
- what passed
- what remains
```
